• And Now The Real Fun Begins!
    Concept:
    1) Users write the story (some combination of writing and recording) using the story elements they rolled.
    2) This process would include multiple phases: storyboarding, drafting, revising, animating, and showcasing.
    3) This could either be done independently, or what might be interesting would be if users were able to write the story together: different models for collaborative story writing would need to be explored (and designed accordingly).
    4) Plenty of ways to add supports: sentence starters, word prediction, many opportunities for supporting decoding/encoding skills.
